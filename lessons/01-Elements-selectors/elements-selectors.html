DOMS

1) Por id de la etiqueta

Por id de la etiqueta

Usando:


console.log('--------- 1 ---------');
console.log(document.getElementById('menu'));

2) Acceder por id, usar #

Consulta de selector, pudiendo ser una clases css o id (usar #)
Es lento a comparación de getElementById por las validaciones a comparar

Usando:


console.log('--------- 2 ---------');
console.log(document.querySelector('#menu'));

3) Trae la primera etiqueta que aparezca

Usando:


console.log('--------- 3 ---------');
console.log(document.querySelector('a'));

4) Trae todas las etiquetas que aparezcan

Usando:


console.log('--------- 4 ---------');
console.log(document.querySelectorAll('a'));

5) Trae la etiqueta con la posición requerida

Usando:


console.log('--------- 5 ---------');
console.log(document.querySelectorAll('a')[1]);

6) Para acceder a una clase css

Usando:


console.log('--------- 6 ---------');
console.log(document.querySelector('.card'));

7) Para acceder a una clase css por posición

Usando:


console.log('--------- 7 ---------');
console.log(document.querySelectorAll(".card")[2]);
